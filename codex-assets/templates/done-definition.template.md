# Done Definition Template

> 用途：每轮迭代结束时填一次，保证“有证据的完成”。

## Functional

- [ ] 关键用户路径可用
- [ ] 加载态/空态/错态/重试完整
- [ ] 权限受限路径符合预期

## Visual

- [ ] 与设计规范一致（颜色、字号、间距、层级）
- [ ] 大屏/网页对应分辨率下不破版
- [ ] 动效节奏与信息可读性平衡

## Performance

- [ ] 首屏时间在预算内
- [ ] 核心交互延迟在预算内
- [ ] 长列表/图表刷新无明显掉帧
- [ ] 无异常内存增长

## Accessibility

- [ ] 键盘可达
- [ ] 焦点可见
- [ ] 语义标签合理
- [ ] 对比度达标
- [ ] 表单错误提示可读

## Tests

- [ ] 单元测试通过
- [ ] 集成测试通过
- [ ] 视觉回归/快照通过
- [ ] 构建通过
