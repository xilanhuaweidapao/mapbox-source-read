# Prompt: 页面搭建（Page Build）

```md
任务：基于给定 Input Pack 实现一个新页面（或大屏模块），要求一次交付到可验收状态。

请按以下顺序输出：
1. 实现计划（分步骤）
2. 代码改动（仅限指定文件）
3. 变更摘要
4. 风险点
5. 验证命令与结果证据

必须遵守：
1. 不修改无关文件
2. 不绕过测试
3. 严格遵循 DesignTokens、PageConfig、ApiResponse、DoneDefinition
4. 完整处理加载态、空态、错态、重试
5. 提供分辨率适配说明（大屏：1920/3840；网页：360/768/1024/1440）
```
